Tema 2. Define los elementos que componen la estructuración de una página web

• Contenedor (wrapper): Es el elemento que envuelve toda la página o una sección de ella. Su función principal es mantener el contenido alineado y centrado, además de facilitar la gestión de márgenes y paddings.

• Cabecera (header): Es la parte superior de la página que suele contener el logotipo, el título del sitio, y, a menudo, el menú principal de navegación. Sirve como la introducción visual y funcional del sitio.

• Contenido (content): Es el área principal donde se muestra la información más relevante, como texto, imágenes, videos y otros elementos. Este es el corazón de la página, donde se encuentra el mensaje o la funcionalidad que se quiere comunicar.

• Menú (menu): Es una lista de enlaces que permite a los usuarios navegar entre diferentes secciones de la página o del sitio web. Puede estar ubicado en la cabecera, en la barra lateral o en el pie de página.

• Pie (footer): Es la sección final de la página que generalmente incluye información como derechos de autor, enlaces a políticas de privacidad, términos de uso y otros enlaces secundarios. Suele ser un área de contacto o información adicional.

• Lateral (sidebar): Es una columna que se encuentra a un lado del contenido principal. Puede contener menús adicionales, enlaces, anuncios, widgets o información complementaria relacionada con el contenido.

• Etiqueta div: Es un elemento contenedor en HTML que se utiliza para agrupar otros elementos y aplicarles estilos o estructurarlos en la página. No tiene significado semántico por sí misma, pero es fundamental para crear layouts y estilos en CSS.



Tema 3. Identifica y describe los elementos que componen la estructuración semántica de una página web.


• < header >: Define la cabecera de una sección o del documento. Generalmente incluye el logotipo, el título, y elementos de navegación.

• < nav >: Representa un bloque de enlaces de navegación. Se utiliza para agrupar los enlaces principales de navegación de la página.

• < main>: Indica el contenido principal del documento. Solo debe haber un elemento < main> por página y debe contener el contenido relevante, excluyendo cabeceras, pies y barras laterales.

• < article>: Representa un contenido independiente y autocontenido, como una entrada de blog, una noticia o un comentario. Puede ser distribuido de manera independiente.

• < section>: Define una sección temática dentro de un documento. Se utiliza para agrupar contenido relacionado y puede contener un encabezado.

• < aside>: Representa contenido relacionado pero no esencial al contenido principal, como una barra lateral o notas adicionales.

• < footer>: Indica la parte inferior de una sección o del documento. Suele incluir información sobre el autor, derechos de autor y enlaces a documentos relacionados.

• < h1>, < h2>, < h3>, etc.: Estas etiquetas de encabezado se utilizan para estructurar el contenido jerárquicamente, donde < h1> es el más importante y se utiliza típicamente para el título principal.

• < figure> y < figcaption>: < figure> es un elemento que encapsula contenido relacionado como imágenes, gráficos o diagramas, y < figcaption> proporciona una descripción o leyenda para ese contenido.

• < time>: Se utiliza para representar información relacionada con fechas y horas, facilitando la indexación por parte de motores de búsqueda y su comprensión.
